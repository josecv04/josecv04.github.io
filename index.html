Featured Projects

“LegoEEG” Modular EEG Headset

Circuits • PCB • Validation

Modular EEG headset with interchangeable hydrogel electrodes and custom low-noise amplifier + control PCBs, designed for fast iteration and realistic electrode–skin conditions.

  • Schematic → PCB: designed the front-end schematic and implemented the layout in Altium with noise-aware routing/grounding.
  • Validation: benchmarked against existing EEG boards; compared signal quality and stability using analysis scripts.
  • Hydrogel interface: fabricated conductive hydrogels and iterated based on electrode–skin impedance and signal quality.

Microneedle Biosensor Systems (NBE Lab)

Microfab • Sputtering • DSP

  • Integration: integrated electrochemical sensors with biocompatible substrates; debugged interface and noise issues.
  • Signal pipelines: built DSP workflows (Butterworth, baseline correction, Savitzky–Golay) to clean biosensor data.

ECG + EPG Signal Analysis System

DSP • Modeling • MATLAB/Python

Multi-signal acquisition and analysis for biomedical pattern recognition: filtering, synchronization, and model-driven comparison across subjects.
